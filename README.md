# tk-tutorial-gtest-gmock

This tutorial is intended to provide an easy start with googletest and googlemock.

## Installation

```{r, engine='bash', count_lines}
git clone https://github.com/google/googletest.git
cd googletest
mkdir build
cd build
cmake ..
make
sudo make install
```
